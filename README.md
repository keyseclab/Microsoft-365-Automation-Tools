# Microsoft 365 Automation Tools
Questo repository contiene script Python per l'automazione di task amministrativi su tenant Microsoft 365 tramite **Microsoft Graph API**.

## Funzionalità
* Esportazione lista utenti in formato Excel.
* Monitoraggio licenze assegnate (In arrivo).
* Analisi log di accesso (In arrivo).

## Requisiti
* Python 3.x
* Librerie: `requests`, `pandas`, `openpyxl`
